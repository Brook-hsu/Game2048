# Game2048
> 打发时间的游戏2048开发
# 置顶项
* 移动时无非有三种情况：
1.相邻两个数至少有一个数为0，无效果；
2.相邻两个数非零且不等，发生***碰撞***；
3.相邻两个数非零且相等，发生***合并***，并且***计分***；

###  变量命名规范
> 被标固定的变量在定义时就被赋值，所以中途不要改变它们的值

* 数组:name[][]
* 行：Line（固定）
* 列：Col （固定）
### 程序步骤
```
1.打印界面（每次打印要有刷新）              （已完成）
2.初始化变量
3.键入方向键
4.对应操作
5.随机位置生成新数
6.记分数
7.判断游戏是否结束（是：结束；否：回到3）
```
